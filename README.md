# ♟️🎲 Dice Chess — Шахматы с кубиком v1.1

Мультиплеерные шахматы, где классические правила дополнены механикой случайного бюджета ходов. Идеально подходит для локальной игры с искусственным интеллектом (Stockfish) или онлайн-партий с друзьями через локальный туннель.

---

# 📂 Структура проекта

```text
.
├── backend/
│   ├── main.py            # FastAPI-сервер (WebSocket)
│   ├── game.py            # Игровая логика, бюджет ходов
│   ├── ai_adapter.py      # Интеграция с Stockfish
│   └── requirements.txt   # Python-зависимости
└── frontend/
    └── index.html         # Веб-клиент (chess.js + интерфейс)
```

---

# 🛠️ Подготовка к запуску

## 1. Установка шахматного движка Stockfish

Для работы искусственного интеллекта необходим установленный движок **Stockfish**.

### Windows

1. Скачайте Stockfish с официального сайта.
2. Распакуйте архив.
3. Укажите путь к файлу `stockfish.exe` в массиве `_SF_CANDIDATES` внутри файла:

```text
backend/ai_adapter.py
```

### Linux (Ubuntu / Debian)

```bash
sudo apt update
sudo apt install stockfish
```

### macOS

```bash
brew install stockfish
```

---

## 2. Установка зависимостей

Перейдите в папку `backend` и установите необходимые библиотеки:

```bash
cd backend
pip install -r requirements.txt
```

---

# 🚀 Запуск игры

## Шаг 1. Запуск сервера

Из папки `backend` выполните:

```bash
uvicorn main:app --host 0.0.0.0 --port 8000 --reload
```

После запуска игра будет доступна по адресу:

```text
http://localhost:8000
```

---

## Шаг 2. Игра с другом через LocalTunnel

Чтобы открыть игру через интернет:

### Установите LocalTunnel

> Требуется установленный Node.js.

```bash
npm install -g localtunnel
```

### Запустите туннель

```bash
lt --port 8000
```

Вы получите ссылку вида:

```text
https://xxxx.loca.lt
```

Отправьте её второму игроку.

> **Важно**
>
> При первом открытии ссылки LocalTunnel попросит ввести ваш внешний IP-адрес. Его можно узнать, например, на сайте **2ip.ru**. Это стандартная защита сервиса от ботов.

# 📜 Правила Dice Chess

## 🎲 Бросок кубика

В начале каждого раунда игрок бросает кубик.

Выпавшее число **от 1 до 6** становится бюджетом ходов на текущий раунд.

---

## ♟️ Выполнение ходов

Игрок может делать **несколько ходов подряд**, пока хватает бюджета.

Фигуры можно двигать в любом порядке.

---

## 💰 Стоимость ходов

| Фигура |                                                           Стоимость |
| ------ | ------------------------------------------------------------------: |
| Пешка  | По количеству пройденных клеток (1 или 2). При взятии — **1 очко**. |
| Конь   |                                                  **Всегда 3 очка**. |
| Слон   |                                    По количеству пройденных клеток. |
| Ладья  |                                    По количеству пройденных клеток. |
| Ферзь  |                                    По количеству пройденных клеток. |
| Король |                         По количеству пройденных клеток (обычно 1). |

---

## ✋ Завершение раунда

Раунд заканчивается:

* автоматически, когда бюджет становится равен **0**;
* вручную — кнопкой **«Завершить ход»**, если игрок хочет сохранить оставшиеся невозможные для использования очки.

---

## 🏆 Победа

Партия заканчивается по классическим шахматным правилам — после постановки **мата** одному из королей.

---

# 🎮 Пример раунда

Выпало **4**.

Возможная последовательность действий:

1. Пешка `e2 → e4`

   * стоимость: **2**
   * осталось: **2**

2. Конь `g1 → f3`

   * стоимость: **3**
   * ❌ ход невозможен

3. Слон `f1 → c4`

   * стоимость: **3**
   * ❌ ход невозможен

4. Пешка `d2 → d3`

   * стоимость: **1**
   * осталось: **1**

5. Король `e1 → e2`

   * стоимость: **1**
   * бюджет становится **0**
   * ход автоматически переходит сопернику.

---

# 🎯 Особенности

* 🎲 Случайный бюджет делает каждую партию уникальной.
* ♟️ За один раунд можно выполнить несколько ходов.
* 🤖 Поддерживается игра против Stockfish.
* 🌐 Можно играть с друзьями через LocalTunnel без настройки роутера.
* 💻 Весь клиент работает прямо в браузере.
